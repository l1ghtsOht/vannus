# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the Praxis AI Decision Engine.

ADRs are short text documents that capture important architectural decisions along with
their context and consequences. See [Michael Nygard's article](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions.html).
